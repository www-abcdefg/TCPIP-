# ARP

# ICMP
* ARP
    * 將IP位址應設為實體位置
    * 發送資料前會先檢查Cache ARP對照表
* RARP
    * 來源端知道自己的MAC位址，不知道IP位址
    * 運作RARP尋找IP位址
    * 在沒有磁碟機與終端機的時候
